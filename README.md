# >> Pushing Google Sites to the Absolute Limit

> A digital marketing portfolio and modular embed repository designed to break the constraints of the Google Sites ecosystem.

I just launched my portfolio, built 100% on Google Sites. Instead of settling for standard templates, I explored how far custom interactive elements can go using pure HTML, CSS, JavaScript, and Cloudflare edge routing.

Everything here is responsive, functional, and built to demonstrate what this platform can actually do when pushed beyond its intended limits.

>> Live Website: https://www.eryc.my.id

---

## == What's Inside (Highlights)

If you want to see what is truly possible on a free Google Site, explore these:

* [>] The Cyberpunk Terminal (About Page)  
  A fully working command-line interface embedded directly into the page  
  - Easter egg: try typing `sudo`

* [>] 16-Bit RPG Dialog Engine (SEO Page)  
  A retro SNES-style interactive dialog system replacing static content  
  - Explains SEO and digital marketing systems interactively

* [>] Optimized Asset Delivery (CDN Strategy)  
  Custom asset loading using Dropbox (`dl.dropboxusercontent.com`)  
  - Bypasses storage limits  
  - Maintains fast loading performance
